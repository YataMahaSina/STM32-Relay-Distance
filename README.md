# STM32-Relay-Distance
STM32 Relay distance merupakan rancangan prototipe dari sistem pengontrolan relay dengan beban 3 unit lampu yang berbeda watt dengan tegangan 1 fasa 220V, menggunakan sensor ZMPT101 dan ACS712 serta dengan output tampilan display pada LCD 16x2 I2C
